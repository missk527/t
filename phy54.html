  1. Because it represents distance, not area

Estimation & Practical Application (36–40)

  1. Count grid squares
  2. Combine half squares to make full ones
  3. ≈8 meters
  4. Because the line is curved (not straight)
  5. Gradient = acceleration, area = distance

Tip: On v–t graphs, gradient = acceleration. On d–t graphs, gradient = velocity. Area under v–t = distance.
